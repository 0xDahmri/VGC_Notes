# Sample Teams

A place to store and explain full team builds. Each entry includes the paste, a brief team summary, and notes on how to play it.

---

## Template

Use this format when adding a new team:

```markdown
## Team Name

**Format:** Regulation H
**Restricted:** [Pokemon]
**Archetype:** [Rain / Sun / Trick Room / Balance / Offense]
**Paste:** [Showdown paste link]

### Core

[2-3 sentence description of the team's win condition and key synergies]

### Lead Options

- **Lead A + Lead B** — when to use this lead
- **Lead C + Lead D** — when to use this lead

### Key Matchups

- vs. Rain: [notes]
- vs. Zacian: [notes]
- vs. Trick Room: [notes]

### Notes

Any other tips, known weaknesses, or tech choices to explain.
```
